# File Manifest for Local Agent Lab Scoreboard

## Purpose and Ownership

### `data/scoreboard.json`
- **Purpose**: Stores the current state of the experiment, including turn counts, completion status, and any other relevant metrics.
- **Ownership**: Qwen (local worker) - Responsible for maintaining and updating this file.

### `docs/dashboard-content-plan.md`
- **Purpose**: Provides a detailed plan for how the dashboard will be structured and displayed.
- **Ownership**: Qwen (local worker) - Responsible for creating and maintaining this file.

### `docs/data-schema.md`
- **Purpose**: Defines the structure of the data used in the experiment, including field names, types, and any constraints or validations.
- **Ownership**: Qwen (local worker) - Responsible for creating and maintaining this file.

### `README.md`
- **Purpose**: Provides an overview of the project, its goals, and how to run it locally.
- **Ownership**: Qwen (local worker) - Responsible for creating and maintaining this file.
